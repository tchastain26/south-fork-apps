# Magic 8-Ball Notes

Created: 2026.04.23
Status: Done

## What It Does

Magic 8-Ball is a browser toy that reveals randomized answers after you ask a question and shake the orb.

## How to Use

Type a question, then ask the orb for an answer. Use Shuffle Answer if you want a fresh read without editing the question.
