# Virtual Pop Notes

Created: 2026.04.23
Status: Done

## What It Does

Virtual Pop is a reusable browser fidget board with tappable bubbles, size presets, and a reset action.

## How to Use

Tap or click bubbles to pop them, switch board sizes if you want more or fewer bubbles, and reset when you want a fresh board.
