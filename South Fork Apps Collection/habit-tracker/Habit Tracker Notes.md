# Habit Tracker Notes

Created: 2026.04.23
Status: Done

## What It Does

Habit Tracker saves up to ten daily habits in localStorage, lets you check them off each day, and tracks current and longest streaks.

## How to Use

Add a habit, check it off when completed, and come back each day. The app rolls to the current date automatically and keeps history in the browser.
