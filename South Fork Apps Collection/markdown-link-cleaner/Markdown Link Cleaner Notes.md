# Markdown Link Cleaner Notes

Created: 2026.04.15
Status: Done

## What It Does

Convert a list of raw URLs into clean Markdown links. It can strip UTM tracking parameters and optionally fetch live page titles via an external proxy.

## How to Use

1. Paste URLs into the input box (one per line).
2. Select options (Strip UTM, Fetch titles, List format).
3. Click "Convert to Markdown".
4. Copy the result.
