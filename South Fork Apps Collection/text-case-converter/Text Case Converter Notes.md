# Text Case Converter Notes

Created: 2026.04.15
Status: Done

## What It Does

A versatile text transformation tool. Instantly convert text between common and obscure cases like camelCase, snake_case, PascalCase, and many more (20+ styles).

## How to Use

1. Enter text in the input box.
2. The cards below will live-update with various case transformations.
3. Click "Copy" on any card to grab the result.
4. You can drag cards to reorder your preferred case formats.
