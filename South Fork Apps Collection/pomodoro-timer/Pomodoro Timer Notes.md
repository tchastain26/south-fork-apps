# Pomodoro Timer Notes

Created: 2026.04.23
Status: Done

## What It Does

Pomodoro Timer runs 25-minute focus sessions, 5-minute short breaks, and a 15-minute long break after every fourth completed focus session.

## How to Use

Start the timer, pause when needed, or reset the current phase. When a phase ends, the app advances automatically and plays a soft browser-generated beep.
