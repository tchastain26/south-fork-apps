# Tip Calculator Notes

Created: 2026.04.23
Status: Done

## What It Does

Tip Calculator splits a restaurant bill by calculating the tip amount, total bill, and per-person total from a bill amount, tip percentage, and party size.

## How to Use

Enter the bill amount, choose a tip percentage manually or with the quick-select buttons, then set the number of people. The results update immediately.
