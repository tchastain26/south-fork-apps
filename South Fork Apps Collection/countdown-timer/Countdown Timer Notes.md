# Countdown Timer Notes

Created: 2026.04.23
Status: Done

## What It Does

Countdown Timer saves up to five event countdowns in localStorage and shows live days, hours, minutes, and seconds until each event.

## How to Use

Enter an event name and target date/time, add the countdown, and leave the app open for live updates. Expired events show "It's time!" in the accent color.
