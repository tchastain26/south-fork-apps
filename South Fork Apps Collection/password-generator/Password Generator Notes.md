# Password Generator Notes

Created: 2026.04.23
Status: Done

## What It Does

Password Generator creates five random passwords at a time with adjustable length, character set options, one-click copy, and a simple strength rating.

## How to Use

Choose a length between 8 and 64, select which character types to include, then generate passwords. Use Copy beside any password to put it on the clipboard.
