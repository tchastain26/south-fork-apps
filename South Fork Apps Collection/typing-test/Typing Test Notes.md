# Typing Test Notes

Created: 2026.04.23
Status: Done

## What It Does

Typing Test gives a fast browser-based typing challenge with timed modes, live WPM, accuracy, and a local best score per mode.

## How to Use

Pick a duration, start typing the prompt, and watch stats update live. When time runs out, the app shows final speed and accuracy.
