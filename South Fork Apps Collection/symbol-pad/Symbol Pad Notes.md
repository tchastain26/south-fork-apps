# Symbol Pad Notes

Created: 2026.04.23
Status: Done

## What It Does

Symbol Pad is a searchable copy tool for frequent symbols with category filters, one-click copy, and a pinned section that stays saved in the browser.

## How to Use

Search or filter symbols, click any tile to copy it, and use the pin button to keep common symbols in the pinned strip at the top.
