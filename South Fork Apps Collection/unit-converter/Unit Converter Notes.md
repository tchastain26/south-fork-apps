# Unit Converter Notes

Created: 2026.04.23
Status: Done

## What It Does

Unit Converter converts live between length, weight, temperature, volume, and speed units with a two-field layout.

## How to Use

Pick a category, choose the source and target units, type a value, and the converted value appears immediately. Swap reverses the selected units.
