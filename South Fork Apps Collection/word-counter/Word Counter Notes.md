# Word Counter Notes

Created: 2026.04.23
Status: Done

## What It Does

Word Counter analyzes pasted or typed text and reports live counts for words, characters, characters without spaces, sentences, paragraphs, reading time, and speaking time.

## How to Use

Paste or type text into the main text area. The stats update automatically. Use Clear to empty the text area and start over.
