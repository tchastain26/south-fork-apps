# South Fork Apps

Public web app directory for [southforkapps.com](https://southforkapps.com). A solo-built home for vibe-coded web apps out of Bluff City, TN.

## Stack

Single-file static site. Tailwind via CDN, Google Fonts, Material Symbols. No build step.

## Deployment

Auto-deployed to Cloudflare Pages from `main`.
